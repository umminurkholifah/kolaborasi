Introduction

In a co-curricular activity, I got an assignment to create a portfolio website. The website was used to show information and the projects that I had made. At first, I felt confused about choosing the right concept and design for the website, but I kept trying to find ideas and worked on it seriously

02

The Process

First, I prepared the design and collected some pictures and information for the website. After that, I started writing HTML and CSS code on my laptop. Sometimes, I made mistakes in the code, so the website did not look neat. I had to check the code carefully and fix the errors one by one.

03

Development

Then, I added colors, images, buttons, and animations to make the website more attractive. I also learned how to arrange the layout properly so the website would look neat on the screen. Although it was a little difficult, I enjoyed the process because I learned many new things about web design and coding.

04

Result

In the end, I successfully finished my portfolio website. I felt happy and proud because I could complete the assignment and gain a new experience in creating a website. From this activity, I learned to be more patient, careful, and creative when working on a project.
